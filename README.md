# FF14 DCT
FF14国服超域传送及超域返回工具

## 名词
FF14： 最终幻想14

超域传送: 跨服务器大区转移服务

超域返回: 跨服务器大区转回服务

排本: 游戏中进行副本匹配时需要匹配其他玩家角色共同进入的过程

## 背景
中国区FF14因为一些原因使用WEB方式进行超域传送服务。

由于陆行鸟大区活跃度较其他大区高，导致其他区服均向陆行鸟大区进行传送。

这也导致了其他区服的排本时间大幅增加。

种种原因导致了超域传送WEB前端增加了限制。当大区人数超载时，前端将无法选择该大区。

后端目前未进行该限制。

后端接口请求限制每账号1分钟一次。

## 功能
1. 超域传送 含 自动重试
2. 超域返回

## 流程
选择浏览器 --> 在浏览器中登录 --> 在应用中选择功能 --> 选择角色等 --> 功能执行 --> 结束

## 隐私
程序代码开源，仅存在遥测(使用量统计)、赞助(或公告)信息存于远端。
遥测不包含任何信息，仅用于统计程序当日使用次数。
程序不直接接触浏览器中用户输入的信息，仅获取用户登录后服务器提供的凭据/Cookies信息做后续步骤。凭据/Cookies信息仅在用户本地使用。
